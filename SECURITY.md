# Security Policy

## Supported Versions

| Version | Supported |
| :--- | :--- |
| 1.0.x (Main) | ✅ Yes |
| < 1.0.0 | ❌ No |

## Reporting a Vulnerability

If you discover a security vulnerability within ComplianceOS, please report it securely:

1. **GitHub Private Security Advisory**: Submit a private advisory via the **Security** tab on the GitHub repository.
2. **GitHub Security Issue**: Open an issue labeled `security` with reproducible steps.

Please include:
- Description of the vulnerability and potential impact.
- Steps to reproduce or proof-of-concept payload.
- Affected API endpoints or Python modules.

Vulnerability reports will be acknowledged within 48 hours with remediation tracking.
